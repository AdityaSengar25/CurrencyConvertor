## 💱 Currency Converter App

A modern, fast, and reliable currency converter built with a focus on **clean UI**, **real-time data**, and **developer-friendly architecture**.

#### Features
* Real-Time Exchange Rates
* Minimal & Intuitive UI
* Fast & Responsive
* Instant Conversion

#### API Integration

This project uses the **Fawaz Ahmed Currency API (v1)** via jsDelivr:

```
https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies/{currency}.json
```

#### Why this API?

* Free and open-source
* No API key required
* Updated daily
* Lightweight and fast CDN delivery


* **Frontend:** React.js
* **API:** Fawaz Ahmed Currency API (jsDelivr CDN)
* **Styling:** Tailwind CSS
